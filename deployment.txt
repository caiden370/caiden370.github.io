# Masblo GitHub Pages deployment

The deployed app is served from the `gh-pages` branch, not directly from `main`.

## Source branch

Use `main` for source code changes:

```powershell
Set-Location C:\Users\caidenkiani\personal\side_projects\caiden370.github.io
git checkout main
git pull origin main
```

## Deploy the app

Deploy from the React app folder:

```powershell
Set-Location C:\Users\caidenkiani\personal\side_projects\caiden370.github.io\spanish-app
npm run deploy
```

`npm run deploy` runs:

1. `npm run build`
2. `gh-pages -d build`

This builds `spanish-app\build` and pushes the static output to the remote `gh-pages` branch.

## Live site

GitHub Pages serves the `gh-pages` branch at:

```text
https://masblo.com
```

The custom domain is set by `spanish-app\public\CNAME` and copied into `build\CNAME` during the build.

## Important notes

- Pushing `main` updates source code only.
- The live app updates only after `npm run deploy` successfully pushes `gh-pages`.
- There is currently no GitHub Actions workflow for automatic deployment.
- After deploying, confirm the remote `gh-pages` branch advanced:

```powershell
Set-Location C:\Users\caidenkiani\personal\side_projects\caiden370.github.io
git fetch origin gh-pages
git --no-pager log --oneline origin/gh-pages -3
```
